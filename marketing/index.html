04 / 04

Private project intelligence

Your engineering context stays workspace-scoped.

Navide's orchestration, workspace state, and private project intelligence run locally. The `.agent-team/` layer belongs to the individual engineer and is excluded from Git.

Local-first is not universally offline. External agent CLIs, model providers, Git hosts, search, MCP servers, and updates may communicate with third parties when used.

# Navide Marketing Prototype

This directory contains the first public-facing prototype for Navide's product story. It is deliberately static, dependency-free, and separate from the Electron application.

## Preview locally

From the repository root:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173/marketing/>.

## Contents

- `index.html` — bilingual landing-page prototype
- `styles.css` — responsive visual system and motion
- `app.js` — language switcher, mobile navigation, and lightweight interactions
- `assets/social-preview.svg` — reusable social-preview artwork
- `positioning.md` — audience, category, vocabulary, and claim ledger
- `demo-storyboard.md` — real-product recording plan
- `founder-case-study.md` — first-party dogfooding proof draft
- `launch-kit.md` — launch narratives, FAQ, and recurring Build Notes format
- `validation-guide.md` — five-second and demo-comprehension test script
